# grandparent-network-security
Securing grandparents home network Using pi-hole and IDS.

### Issue: 
Grandparents are reckless on the internet and download files/click on ads not knowing what they are doing.

### Solution: 
Install Pi-hole and Suricata on an HP EliteDesk, installed in-line between modem and router.

### Hardware:
HP EliteDesk
Usb -> Ethernet adapter
